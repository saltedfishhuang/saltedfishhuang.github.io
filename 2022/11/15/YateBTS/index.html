YateBTS

Posted on 2022-11-15
  1. 安装Yate以及YateBTS

svn checkout http://yate.null.ro/svn/yate/trunk yate
svn checkout http://voip.null.ro/svn/yatebts/trunk yatebts

Yate

cd /yate
./autogen.sh
./configure –prefix=/usr/local

make

sudo make install
sudo ldconfig

YateBTS

cd /yatebts
./autogen.sh
./configure –prefix=/usr/local

make

sudo make install
sudo ldconfig

Web GUI(It’*s necessary to install apache2 and php)*

sudo touch /usr/local/etc/yate/snmp_data.conf /usr/local/etc/yate/tmsidata.conf

sudo chmod g+w /usr/local/etc/yate/*.conf

cd /var/www/html

ln -s /usr/local/share/yate/nib_web nib

chmod -R a+w /usr/local/etc/yate

  1. Configuring

通过浏览器访问 your-ip/nib 进行配置

如果需要GPRS上网，还需要配置数据转发

iptables -t nat -A POSTROUTING -o ens33 -j MASQUERADE
