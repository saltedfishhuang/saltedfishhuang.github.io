一、区块(Block)

区块是构建区块链的基本单位。一个区块至少要包含以下信息：

  • Index：区块在区块链中的位置；
  • timestamp：区块产生的时间；
  • transaction：区块包含得交易；
  • previousHash：前一个区块得Hash值；
  • Hash：当前区块的Hash值；

区块链不可篡改特征有最后两个属性保证，previousHash和Hash是区块链的关键
