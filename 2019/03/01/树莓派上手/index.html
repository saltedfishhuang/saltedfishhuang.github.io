树莓派上手

Posted on 2019-03-01 Edited on 2020-01-04

首先感谢富婆zz

很久以前就打算买一个树莓派体验一下,一直也拖着没买，前几天在群里说了一下想买，富婆zz直接甩出一句，你要用的话，把我的拿过去就行。emmmmm，留下了贫穷的泪水，于是我就开始了我s的树莓派白嫖之旅。

上手体验

这次拿到的树莓派是Raspberry Pi 3 Model B，拿到手时，拆开包装还收那种电子器件新鲜的味道，基本就是全新的emmmmm

安装系统及配置

镜像下载

拿到这个树莓派时只有一个主机，需要自己哪内存卡重新装一个系统，于是我到官网上一次性下载了3个版本，防止装的时候出问题（立下flag），我下载的三个版本，从上到下依次为raspbian桌面+常见应用版，最简纯命令版，NOODS版。

镜像烧录

我没有使用官方推荐的软件烧录到内存卡上，而是使用的Rufus 3.4.1430，这个软件不用安装，体量也很小，平常装系统烧录镜像我也是用的这个，非常方便！！！

没事千万不要立flag，在烧录镜像时就遇到了问题，下载的ROODS的包根本就无法打开，只有

raspbian的两个镜像可以正常烧录，但是在烧录好后安装过程中又遇到了问题。

完整版的系统烧录好后插入树莓派通电开机后，正常进入了系统，正常配置联网，修改pi的密码后，提示重启，这些都没有问题，可重启后进入用户登录界面后，输入密码不管怎样都是密码错误，即使直接修改密码后重新登陆也提示密码错误，经过多次尝试后，最后放弃了这个完整带有桌面的版本。

最终下载的3个镜像只有最小的那个，纯命令的系统安装成功，并成功激活了root账号，安装了vim，重新配置了键盘模式 ，还开启了ssh。树莓派在现在的最简包中也可以直接开启ssh了，只需要输入sudo raspi-congig

然后按图片中的选择打开SSH就好。

最后顺便还换了一个中科大的源，毕竟本来的源那速度大家都懂emmmmm

然而并没有成功

刚我以为把上面的东西都弄好，我SSH也能连上了，一切都可以使用的时候，一个重启开机还是登陆失败，最终我才确认，是我那张古董级别的内存卡的问题，于是赶紧某宝上下单了一张Class10 的闪迪32G的内存卡，重新将带桌面的完整镜像烧录进去，开机、配置、使用，现在都没有问题，接下来要做的事就是要好好学习了，不能落后其他人太多了。那张古董内存卡的问题，有时间再去好好看看吧！（20190304）
